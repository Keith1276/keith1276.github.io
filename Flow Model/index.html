模型思想

回顾一下生成模型的核心目标：最大观测数据的对数似然

然而，直接对高维复杂数据的对数似然进行建模和优化很困难，于是像VAE和Diffusion方法，往往通过优化对数似然的一个下界（ELBO）来间接达到目的。虽然在实践中取得了成功，但是始终和直接优化存在一定差距。

基于流的模型提供了一条直接优化对数似然的路径，核心思想在于构建一个从真实数据分布到一个简单、易于处理的先验分布的可逆映射。借助数学中的变量替换定理，这个可逆映射可以将复杂数据分布下的概率密度计算转化为简单分布下的概率密度计算，从而实现对数似然的直接建模和优化。

流模型的关键点

Key point1: 流模型在这两个分布中数据维度是严格一致的，与VAE不同，这使得能够保留数据完整的信息。

**Key point2：**参数共享映射，无论正向过程还是反向构成，共享同一组参数，而且优化直接是对数似然。

所以设计流模型必须有如下关键要求：

  • 必须可逆
  • 雅可比矩阵必须易于计算

![image-20251020094118292](./Flow Model/image-20251020094118292.png)

由于单个可逆变换的表达能力通常有限，特别是还要满足可逆、雅可比矩阵易于计算的约束，实践中通常串联多个可逆变换，实现更强大的复合变换。

![image-20251013074835217](./Flow Model/image-20251013074835217.png)

Coupling Blocks

为了满足flow 模型可逆并且雅可比矩阵易于计算的要求，有多种设计，其中放射耦合层是很多工作的核心组件。

他的基本思想是将输入向量分成两个部分，变换的时候，一个保持不变，另一个通过放射变换进行更新，满足这个规则的Jacobian会非常好算，示意图如下：
